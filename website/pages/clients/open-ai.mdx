# OpenAI

<Callout type="info">
  You need a valid OpenAI API key to use this client. You can get one
  [here](https://platform.openai.com/account/api-keys).
</Callout>

The OpenAI `client` handles the communication with OpenAI's API, providing a way for a `prompter` to send requests to the API.\
Under the hood it simply uses a native `fetch` call to make HTTP requests to the API and get the responses in plain text.

## Basic usage

```typescript copy
import { OpenAiClientBuilder } from "gptyped"

const gpTypedOpenAiClient = new OpenAiClientBuilder("YOUR_OPEN_AI_SECRET_KEY").build()
```

_**Note:** replace `YOUR_OPEN_AI_SECRET_KEY` with your actual [OpenAI key](https://platform.openai.com/account/api-keys)._
